Project of simple handheld gaming console made using Proteus and Keil software.  
Project schematic consists of 4 AT89C52 microcontrollers connect via RS485/RS422 bus.
Each microcontroller controls one component of the console (display, matrix keyboard, 7 segment display, small lights and motor).
